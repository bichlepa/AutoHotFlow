Translation tool help

This documentation is designed for those who want to add translations of AHF.

You need some tools in some steps. You'll find them here:

  • Translation tool: language\Translation Tool.ahk
  • Find modules: find modules.ahk

How to start

This section describes how you can create new elements with minimum effort.

  • Open the Translation tool
  • Select the workspace. There are translation files for AHF itself and for each package. AHF comes with the "Default" package.
  • If you want to add a new language
    • Click on "Add new language"
    • Fill out the form and click on OK
  • Select an entry in the tree view (on left side) and the language in the list view (on right side)
  • Enter the translated string. The tool saves the changes instantly. You don't need to think about saving at all.
  • Proceed with all other entries.
  • Each element has a help files. You'll find them in Source_Elements\Default\help.
  • Copy the help files to a folder that matches the ID of your language, open the html files and translate them.
  • Run the tool Find modules. This will make your changes visible to AHF.
  • Reload AHF and check the result.
  • Send the all translated files via email to autohotflow@arcor.de. I'll include it in the next update of AHF.
  • Please update your translations regularely. If AHF gets some updates and the translations won't get any update, I'll remove it if the amount of untranslated strings gets too big.

Remarks

You can highlight untranslated entries: Select the language in the dropdown and click on "Find untranslated entries". Now all untranslated entries will be highlighted.

You can highlight entries with specified string: Select the language in the dropdown, write search text in the edit field and click on "Search". Now all entries with that string will be highlighted.

If you encounter any problem, please contact via GitHub Issues (preferable), via forum or via autohotflow@arcor.de.
