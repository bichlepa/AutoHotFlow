Read from file

The action Read from file reads the contents of a file.

Settings

Output variable name

Name of the variable which will contain the result.

File path

Name and path of the file where the download will be written.
The file path is assumed to be in the working directory if an absolute path isn't specified.

Encoding

The text in the files may be encoded differently. If the encoding is not set correctly, some of the characters can't be read properly.

Linefeed

Standard text files in windows use a linefeed and a carriage return when there is a line break. It might be useful to replace the two characters by a single linefeed.
