Loop through files

The loop Loop through files retrieves the specified files or folders, one at a time.

Settings

File pattern

The name of a single file or folder, or a wildcard pattern.
Both asterisks and question marks are supported as wildcards. A match occurs when the pattern appears in either the file's long/normal name or its 8.3 short name.
The file pattern is assumed to be in the working directory if an absolute path isn't specified.

Options

If Operate on files is checked, only files that match the pattern will be found.

If Operate on files and folders is checked, files and folders will be found.

If Operate on files is checked, only folders will be found.
If Recurse subfolders into is checked, all files and folders contained therein are retrieved if they match the file pattern. All subfolders will be recursed into, not just those whose names match the file pattern.

Supplied variables

A_Index

contains the number of the current iteration. It starts with the value 1 and increases on every loop.

A_LoopFileName

contains the name of the file or folder currently retrieved (without the path).

A_LoopFileFullPath

contains the path and name of the file/folder currently retrieved.
Any short folder names in FilePattern will still be short (see next item to get the long version).

A_LoopFileLongPath

contains the long path and name of the file/folder currently retrieved.

A_LoopFileLongPath

The 8.3 short path and name of the file/folder currently retrieved.
For example: C:\MYDOCU~1\ADDRES~1.txt.

A_LoopFileShortName

The 8.3 short name, or alternate name of the file. If the file doesn't have one A_LoopFileName will be retrieved instead.

A_LoopFileDir

The path of the directory in which the found file resides.

A_LoopFileTimeModified

The time the file was last modified.

A_LoopFileTimeCreated

The time the file was created.

A_LoopFileTimeModified

The time the file was last modified.

A_LoopFileTimeAccessed

The time the file was last accessed.

A_LoopFileAttrib

The attributes the file currently retrieved.
The string will contain a subset of the letters in the string "RASHNDOCT":

R = READONLY
A = ARCHIVE
S = SYSTEM
H = HIDDEN
N = NORMAL
D = DIRECTORY
O = OFFLINE
C = COMPRESSED
T = TEMPORARY

A_LoopFileSize

The size in bytes of the file currently retrieved. Files larger than 4 gigabytes are also supported.

A_LoopFileSizeKB

The size in Kbytes of the file currently retrieved, rounded down to the nearest integer.

A_LoopFileSizeMB

The size in Mbytes of the file currently retrieved, rounded down to the nearest integer.

A_LoopCurrentList

contains a list, which contains all informations about all files.
