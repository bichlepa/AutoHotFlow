Conditional loop

The loop Conditional loop allows to repeat some actions several times. It will continue iteration until the condition is true.

Settings

Condition

An expression which will be checked on each iteration.

If Evaluate on first iteration is checked, the condition will be checked before first iteration. If it is not checked, the first iteration will be always performed, with no regard whether the condition is true or not.

Examples:

  • (1 or 0) and 1 (results to 1)

  • !(5 < 3) (results to 1)

  • num > 5 && num <= 7 (true if num contains a number between 6 or 7)

  • a_index <= 5 (performs 5 iterations)



Supplied variables

A_Index

contains the number of the current iteration. It starts with the value 1 and increases on every loop.
