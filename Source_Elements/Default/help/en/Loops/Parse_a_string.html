Parse a string

The loop Parse a string retrieves substrings (fields) from a string, one at a time.


Settings

Input string

If This is a string is selected, this field will contain the input text. You may insert a variable's content by enclosing the variable in percent signs.

If This is a variable name or expression is selected, this field will contain the variable name.

Delimiter characters

This field contains the separator characters (case sensitive). On every appearance of one of those characters the input text will be split into substrings.

If this field is blank, each character of the input string will be treated as a separate substring.

You can use a space, tabulator and linefeed as delimiters by using following variables:

a_space, a_tab, a_lf

Note!

Each letter is a delimiter. Don't separate the delimiters by a comma or space, unless you want the comma or space to be a delimiter.

Examples:

  • , (Use comma as delimiter)

  • %a_space%%a_tab%%a_linefeed% (Retrieve each word in a text)

  • %delimiters% (variable that contains the delimiters)

  • ab c (use the letters a, b, c and a space as delimiter)



Omit characters

This field contains the characters (case sensitive) to exclude from the beginning and end of each substring.

If this field can be blank.

You can use a space, tabulator and linefeed by using following variables:

a_space, a_tab, a_lf

Supplied variables

A_Index

contains the number of the current iteration. It starts with the value 1 and increases on every loop.

A_LoopField

contains the current substring.

A_LoopCurrentList

contains a list, which contains all substrings.
