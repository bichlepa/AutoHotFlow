Periodic timer

The trigger Periodic time launches the flow periodically.

Settings

Time interval

Specify the time interval. Don't for get to specify the unit!

Supplied variables

A_ThisHokey

contains the text (if any) which were found in the clipboard.

A_ContentType

contains one of following values:
- 0 if the clipboard is now empty
- 1 if it contains something that can be expressed as text (this includes files copied from an Explorer window)
- 2 if it contains something entirely non-text such as a picture

Remarks

It will not trigger on activation, instead the entirety of its period must elapse before it triggers the flow.

The maximum amount of periodic timers in one flow is 10.

Do not specify a period smaller than 100 ms. AutoHotFlow can't work that fast.
