User idle time

The trigger User idle time launches after the user was inactive for a specified amount of time.

Settings

Period of time

The flow will start after the specified period of time.

Method

Select, which method should be used to find out the user idle time.
Method 1: Every user input and simulated user input (e.g. using action Send keystrokes)
Method 2: Only physical user input. Simulated input will be ignored. It might be, that the physical mouse input will be ignored, too.
