Clipboard changes

The trigger Clipboard changes launches the flow whenever any application has changed the contents of the clipboard.

Settings

Options

If Trigger only if the clipboard contains some text is checked, it will not trigger if the clipboard contains something entirely non-text such as a picture.

Supplied variables

A_Clipboard

contains the text (if any) which were found in the clipboard.

A_ContentType

contains one of following values:
- 0 if the clipboard is now empty
- 1 if it contains something that can be expressed as text (this includes files copied from an Explorer window)
- 2 if it contains something entirely non-text such as a picture

Remarks

If the clipboard contains files that were copied using Explorer, it will contain a string containing the full paths of the files, one in each line.

Note!

Only one of this trigger can be activated at once.
