Shortcut

The trigger Shortcut creates a shortcut file which can be launched to trigger the flow.

Settings

File path

Name and path of the shortcut file. The file path is assumed to be in the working directory if an absolute path isn't specified.

Options

If Remove shortcut when disabling is selected, the shortcut will be deleted, as soon as the flow disables or exits.
If it is not selected, the shortcut will remain. It will always be possible to launch the flow, even if it is not activated. But still the AutoHotFlow manager needs to run.
