List contains element

The condition List contains element checks whether a list contains an item.

Settings

Input list

The input list.

Search for what

It's possible either to search for an index or key or to search for an element with a specific content.

Key or index

Only accessible if Search for an index or key is selected.
Specify the position or key of the value to retrieve.

Seeked content

Only accessible if Search for an element with a specific content is selected.
Enter the value to search for.

Supplied Variables

A_FoundKey

This variable is only set if Search for an element with a specific content is selected.
It contains the found key name which contains the searched content.
