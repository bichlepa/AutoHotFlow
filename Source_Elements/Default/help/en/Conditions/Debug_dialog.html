Expressions

The condition Expression evaluates an expression and decides whether the result is true or false. This is similar to an if-statement.

Settings

Expression

Expression to evaluate.
The condition exits with “yes” if the result of the expression is 1 or an other value without 0.
The condition exits with “no” if the result is 0 or empty.

Examples:

(1 or 0) and 1 (results to 1)

number > 5 && number <= 7 (results to 1 if number is between 6 or 7)

Var%VarContainingTwo% (Accessing variable Var2)
