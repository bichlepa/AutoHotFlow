Search in a string

The action Search in a string searches for the position of the specified substring in a string.

Settings

Output variable

Name of the variable which will contain the result. This is the position of the substring.

Input string

A string whose contents will be read from.

Text to search

The string to search for.

Which occurrence

Defines which occurrence from left or right will be found.

Start position

Defines the offset from left or right where the search begins. Position 1 is the first character.

Case sensitivity

If Case insensitive is selected, the character case will be ignored. For example if the input string is “Hello World”, the search text “world” will match.
If Case sensitive is selected, the character case will not be ignored.
