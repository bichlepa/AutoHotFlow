Get file attributes

The action Get file attributes retrieves the attributes (read-only, hidden, etc) of a file.

Settings

File path

Name and path of the file. The file path is assumed to be in the working directory if an absolute path isn't specified.

New time

The timestamp to set. The timestamp has the format YYYYMMDDHH24MISS.

Which time.

Which time to change.

Options

This action can perform on files, forlder or both. It can also iterate in the subfolders.
