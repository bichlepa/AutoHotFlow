Set file time

The action Set file time sets a datetime stamp of a file or folder.

Settings

New time (Date format)

Name of the variable which contains a date.

Source file

The name of a single file or folder, or a wildcard pattern.

Both asterisks and question marks are supported as wildcards. A match occurs when the pattern appears in either the file's long/normal name or its 8.3 short name.
The file pattern is assumed to be in the working directory if an absolute path isn't specified.

Which time

Specify the which file time should be changed.

Options

If the time of more than one file or folder should be changed, use a wildcard in Source file and modify following options:

If Operate on files is checked, only files that match the pattern will be found.

If Operate on files and folders is checked, files and folders will be found.

If Operate on files is checked, only folders will be found.
If Recurse subfolders into is checked, all files and folders contained therein are retrieved if they match the file pattern. All subfolders will be recursed into, not just those whose names match the file pattern.
