Convert list to string

The action Convert list to string converts a list to a string. Supported formats are JSON and YAML.

Settings

Output variable name

Name of the variable which will contain the string.

Input list

The input list.

Output format

Specify the format of the string.

Options

If JSON is selected, some options can be set.

If Compact mode is selected, the output string will not have any linebreaks or indentation. Otherwise the intentation length can be set.
