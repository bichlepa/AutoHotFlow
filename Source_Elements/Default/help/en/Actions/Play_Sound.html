Play sound

The action Play sound plays a system sound, or any sound file of supported type.

Settings

Select sound

There is a selection of standard system sounds.
If Sound file is selected, specify the name and path of the file. The file path is assumed to be in the working directory if an absolute path isn't specified.

Preview

You can start and stop the sound

The sound playback will not automatically stop when you close the element settings window.

Remarks

Only one sound can be played simuntaneously. If this action is executed while an other sound is playing, it will stop the playback and play the new sound.
