New list

The action New list adds a value to a list. If the list does not exist, it will be created.

Settings

Variable name

Name of the variable which will contain the list.

Number of elements

The new list can be empty.
Optionally one or multiple elements can be initially added to the list.

Content to add

If Initialize with one element is selected, this field can contain either a value or an expression.

If Initialize with multiple elements is selected, this multiline field can contain one or multiple values that should be added. Specify which delimiter will be used. The default setting is a new element in each line.

Where to insert

Specify the position or key of insertion.

If First position is selected, the value(s) will be added numerically as the first item(s) with the index 1 (, 2, …). The indexes of other items will be incremented.

If Last position is selected, the value(s) will be added as the last item(s). For example if a list has three elements, the other element(s) will be added as the fourth element (, fifth element, …)

If Following position or key is selected, the value(s) will be added into a specified numerically position. If only one element will be added, any key name can be specified.
