Date calculation

The action Date calculation either adds some amount of time to a date or calculates the differences between two date.

Settings

Output variable name

Name of the variable which will contain the result.

Input variable name

Name of the variable which contains a date.

Operation

If Add a period is selected, an amount of time will be added or subtracted to a date and a new date will be stored into the output variable.
If Calculate time difference is selected, the difference between two dates will be calculated.

If Add a period is selected, this field contains the amount of time units to add. Specify a negative number in order to subtract an amount of time.

If Calculate time difference is selected, this field contains the name of the variable that contains a second time.

Which unit

Select the time unit.
