Select file

The action Select file displays a standard dialog that allows the user to open or save file(s).
Note!

This action is experimental.

While the dialog is shown, all other threads and instances are stopped. It may happen a the trigger fails to start a new instance.

Settings

Output variable name

Name of the variable which will contain the result.
If Allow to select multiple files is selected, the result will be a list instead of a string.

Prompt

Text displayed in the window to instruct the user what to do.

Root directory

The name of a single file or folder. This file or folder will be the preselected file or folder when the dialog appars.
The file pattern is assumed to be in the working directory if an absolute path isn't specified.

Filter

Indicates which types of files are shown by the dialog.

Examples:

All Files (*.*)
Documents (*.txt)
Audio (*.wav; *.mp2; *.mp3)
