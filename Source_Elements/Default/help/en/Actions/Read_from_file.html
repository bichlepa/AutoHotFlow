Read from file

The action Read from file reads the content of a file.

Settings

Output variable name

Name of the variable which will contain the result.

File path

Path of the file which contents will be read.
The file path is assumed to be in the working directory if an absolute path isn't specified.

Encoding

The text in the files may be encoded differently. If the encoding is not set correctly, some of the characters can't be read properly.

Size limit

The amount of read data can be limited.

Linefeed

Standard text files in windows use a carriage return & linefeed when there is a line break. It might be useful to replace the two characters with a single linefeed.
