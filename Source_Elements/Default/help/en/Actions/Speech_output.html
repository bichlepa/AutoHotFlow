Speech output

The action Speech output uses an installed text to speech engine to read some text.

Settings

Text to speak

Enter the text that should be spoken. A variable's content can be inserted by enclosing them into percent signs.

Wait settings

If Wait until previous speech output has finished is selected, the flow will pause if the text to speech is currently used by an other element of the same flow. Otherwise either the current speech output will be interrupted and a new one will be started.

If Wait until current speech output finishes is selected, the flow will pause until the speech output will finish.
Note!

If Wait until current speech output finishes is selected, all other threads (if any) of the same flow will stop, too. It won't affect other flows.



Speech engine

This list contains all installed speech engines of your computer. Select one of them.

AutoHotFlow tries to preselect a speech engine that has the same language as the OS language.

Volume

Specify the volume from 0 to 100 %.
A volume of 100 % means that it will be spoken with the current sound volume.

Speed

Specify the speed of the speech.

Pitch

Specify the pitch of the speech.
