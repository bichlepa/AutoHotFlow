List drives

The action List drives lists all drives in the system.

Settings

Output variable name

Name of the variable which will contain the list of found drives.

Drive type

If Find all drives is selected, all drives types are retrieved.
If Get only a specific type of drive is selected, you can choose which type of drive are retrieved.

If no drive can be found

Choose whether the variable will be empty or this action should throw an exception if no drive can be found.
