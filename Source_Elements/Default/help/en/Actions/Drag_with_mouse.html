Drag with mouse

The action Drag with mouse dlicks and holds the specified mouse button, moves the mouse to the destination coordinates, then releases the button.

Settings

Which button

Specify the button to click.

Position, Start coordinates and End coordinates

The x/y coordinates of the drag's start and end position.

Method

There are different methods to simulate mouse events. If the mouse click is not recognized by the application using the default method, you may try change it.
Input mode may be not available if an other flow has hotkey triggers that use some specific hotkeys. If so, the event mode will be used instead.

Speed

If Move mouse before clicking is selected, the mouse movement speed can be changed here.
If Input mode or Play mode is selected, the mouse will always move instantly.
