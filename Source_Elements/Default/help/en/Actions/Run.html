Run

The action Run executes an external program.

Settings

Target

The name of a single file, document, URL that will be opened.
If this field is a file and an absolute path isn't specified, the file pattern is assumed to be in the working directory.

Value

If Replace variables is checked, the variable names which are enclosed in percent signs are replaced by their value.

Examples:

Run Notepad.

Target: notepad

Run VLC media player.

Target: C:\Program Files\VideoLAN\VLC\vlc.exe

Open a document.

Target: %A_MyDocuments%\
Check: Replace variables

Open a website.

Target: http://www.autohotkey.com

Supplied variables

A_PID

contains the Process ID of the launched process.
