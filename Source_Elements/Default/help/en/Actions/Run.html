Run

The action Run runs an external program.

Settings

Command

The name of a single file, document, URL that will be opened.
If an absolute path isn't specified, the file pattern is assumed to be in the working directory.

Working directory

The working directory of the executed application.
If an absolute path isn't specified, the file pattern is assumed to be in the working directory.
In many cases this parameter does not make a difference.

Run mode

Some applications can be started in a specific mode. This can be selected here.

Supplied Variables

A_PID

Process ID of the launched process.

Examples

Run Notepad.

Command

notepad

Run VLC media player.

Command

C:\Program Files\VideoLAN\VLC\vlc.exe (The VLC Media player may be installed in an other location)

Open a document.

Command

%A_MyDocuments%\exampleDocument.odt

Open a website.

Command

http://www.autohotkey.com
