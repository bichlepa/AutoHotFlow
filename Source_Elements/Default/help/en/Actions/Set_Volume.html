Set volume

The action Set volume sets the master volume of the first sound device.

Settings

Action

Specify which action should be done.

Mute settings

If Set mute state is selected, specify which mute state to set.

Absolute volume

If Set a specific volume is selected, specify the new volume from 0 to 100 %.

Relative volume

If Increase or decrease volume is selected, specify by how many percent the volume should be increased or decreased.
