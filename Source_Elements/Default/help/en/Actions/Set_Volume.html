Set volume

The action Set volume gets the master volume of the first sound device.

Settings

Action

Specify which action should be done.

Mute settings

If Set mute state is selected, specify which mute state to set.

Absolute volume

If Set a specific volume is selected, specify the volume from 0 to 100 %.

Relative volume

If Increase or decrease volume is selected, specify how many percent the volume should increase or decrease.

Supplied variables

A_SoundVolume

contains the current sound volume (0 to 100).
