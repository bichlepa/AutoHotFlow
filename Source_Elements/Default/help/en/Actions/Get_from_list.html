Get from list

The action Get from list retrieves a value from an existing list.

Settings

Output variable name

Name of the variable which will contain the value.

Input list

Name of the variable which contains the list.

Key or position

Specify the position or key of the value to retrieve.
If First position is selected, the first value (usually with the key 1) will be retrieved.
If Last position is selected, the first value with the highest key will be retrieved.
If Following position or key is selected, the specified key will be retrieved.
