Set lock key

The action Set lock key sets the state of one of the lock keys.

Settings

Which lock key

Specify which lock key should be changed.

Status to set

Specify whether the flow should be enabled or disabled.

If On or Off is selected, the key is turned on or off.
If Toggle is selected, the key state will be toggled.
If Always on or Always off is selected, the key is turned on or off and forces the key to stay in this state permanently.
