Rounding a number

The action Rounding a number finds the next integer number to another number or cuts off some digits after comma.

Settings

Output variable name

Name of the variable which will contain the result.

Input number

A number, variable or expression.

Examples:

  • -1.3

  • number (Variable that contains a number)

  • 2 - number



Places after comma

A number, variable or expression.

If this field contains 0, the result will be an integer. Otherwise the places after comma will be shortened.

Operation

If Round normally is selected, the nearest integer will be found.
It is also possible to round up or to round down.
