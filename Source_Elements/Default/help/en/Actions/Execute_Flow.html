Execute flow

The action Execute flow executes a flow. The target flow must be enabled.

Settings

Flow name

Enter the flow name which should be executed.

Options

If Send local variables is checked, all local variables that were set in the current instance will be sent to the executed flow.
If Skip disabled flow is checked, the action will not cause an error if the target flow is not enabled.
If Wait for called flow to finish is checked, the action will wait until the target flow has finished.
If Return variables to the calling flow is checked, all generated variables in the target flow will be sent back after it finishes.
