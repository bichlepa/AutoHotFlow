Set monitor settings

The action Set monitor settings changes some attributes of a monitor.

Settings

Which monitor

Either all or one of the monitors can be set.
If your computer has only one screen, this option has no effect.

Brightness

If Change brightness is checked, the brightness will be set to the specified value.

Contrast

If Change contrast is checked, the contrast will be set to the specified value.

Color gain

If Change red gain, Change green gain or Change blue gain is checked, the gain value of the chosen color will be specified. For example if the red value is higher than the other values, the screen will become reddish.
Note!

Be careful. If specifying the value 0 for each color, some monitors might become black.



Color drive

If Change red drive, Change green drive or Change blue drive is checked, the drive value of the chosen color will be specified.
The color drive is similar to the color gain but somehow different.

Color temperature

If Change color temperature is checked, the color temperature will be set to the specified value.
If color temperature and color gain are set in one action, first the color gain and drive will be set and then the color temperature. Mostly the color temperature is a preset for each color gain and drive, therefore it will not make sense to set the color gain and drive and the color temperature in one action.

Remarks

Not every monitor supports those settings.
This action requires Windows Vista or later.
