Recycle file

The action Recycle file sends a file or directory to the recycle bin, if possible.

Settings

File pattern

The name of a single file or folder, or a wildcard pattern.
Both asterisks and question marks are supported as wildcards.
The file pattern is assumed to be in the working directory if an absolute path isn't specified.

To recycle an entire directory, provide its name without a trailing backslash.

Examples:

Recycle one file from MyDocuments folder.

File pattern: %A_MyDocuments%\Example.txt

Recycle all files from a drive.

File pattern: G:\*

Recycle all text files from the working directory.

File pattern: *.txt
