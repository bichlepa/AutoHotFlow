Get monitor settings

The action Get monitor settings retrieves some attributes of a monitor.

Settings

Which monitor

Select a montor.
If your computer has only one screen, specify 1.

Output variables

Name of the variables which will contain the retrieved settings. If a attribute is not needed, the corresponding field can be left empty.

  • Brightness, contrast, red gain, green gain, blue gain, red drive, green drive, blue drive: Those three variables (for each setting) will contain the current value, the minimum value and the maximum value.

  • Color temperature: This variable will contain the current color temperature. For example: 4000K

Remarks

Not every monitor supports those settings.
This action requires Windows Vista or later.
