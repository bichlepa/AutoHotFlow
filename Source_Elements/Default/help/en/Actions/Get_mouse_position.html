Get mouse position

The action Get mouse position simulates a click with any mouse button. The coordinates can be specified.


Settings

Output variable X position and Output variable Y position

Name of the variables which will contain the result.

Mouse position

Specify whether the retrieved mouse position should be relative to the position of the screen, active window or the window client.

Supplied variables

A_WindowID

contains the unique ID of the window that is currently covered by the mouse.

A_ControlID

contains the unique ID of the control that is currently covered by the mouse.
