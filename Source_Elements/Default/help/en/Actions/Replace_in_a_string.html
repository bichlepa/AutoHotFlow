Replace in a string

The action Replace in a string replaces the specified substring with a new string.

Settings

Output variable name

Name of the variable which will contain the result.

Input string

The input string.

Text to search

The string to search for.

Replace by

Found text will be replaced with this text.
If blank, the found text will be deleted.

Number of replacements

Either one occurrence other all occurrences can be replaced.

Case sensitivity

If Case insensitive is selected, the character case will be ignored. For example if the input string is “Hello World”, the search text “world” will match.
If Case sensitive is selected, the character case will not be ignored.

Supplied Variables

a_numberOfReplacements

Number of replacements that were performed.
