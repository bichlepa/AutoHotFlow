Empty recycle bin

The action Empty recycle bin empties the recycle bin.

Note!

Be careful using that action. It may cause loss of data.

Settings

Which drive

If All files is selected, the recycle bin for all drives is emptied.
If Specified drive is selected, only the recycle bin for the specified drive is emptied. The drive is specified by its letter.

Examples:

Empty recycle bin for the drive C.

Select Specified drive
Enter: C:\
