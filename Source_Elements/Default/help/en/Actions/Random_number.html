Random number

The action Random number creates a new variable. The output variable will be of same type as the input variable.

Settings

Variable name

Name of the variable which will contain the copy.

Minimum value and Maximum value

A number, variable or expression.
If both numbers are integers, the generated number will be an integer. If one of them is a floating point, the generated number will be a floating point.

Examples

Get an integer between 1 and 10.

Minimum value

1

Maximum value

10

Get a floating point number between -10 and 10.

Minimum value

-10.0

Maximum value

10.0
