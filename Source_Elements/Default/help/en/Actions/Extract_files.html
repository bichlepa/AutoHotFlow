Extract files

The action Extract files extracts files from an archive.
Following formats are supported: zip, 7z, xz, tar, gzip, BZIP2, Z, lzma

Settings

Source archive

Path of the archive file.

Destination folder

Folder where the files from the archive will be extracted.

Options

Format: By default the format of the archive will be automatically determined (*), however you may choose the expected format.

Remarks

This action uses the command line version of 7zip.
